App pojistenci


Tato aplikace je webová aplikace pro správu a prodej pojištění. 
Uživatelé se mohou přihlásit nebo zaregistrovat, aby mohli prohlížet, kupovat a spravovat svá pojištění. 
Aplikace také umožňuje administrátorům spravovat uživatele a produkty.

Instalace
Naklonujte si projekt z GitHubu:
bash
Copy code
git clone https://github.com/vaserepo/nazev-aplikace.git
Nainstalujte potřebné balíčky:
Copy code
pip install -r requirements.txt
Spusťte aplikaci:
Copy code
python app.py
Použití
Pro použití aplikace otevřete svůj webový prohlížeč a přejděte na adresu http://localhost:5000. Zde se budete moci přihlásit nebo zaregistrovat, aby vám byly přístupné další funkce aplikace.
